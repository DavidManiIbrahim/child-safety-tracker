# Child Safety Tracker

## Project info

A comprehensive child safety tracking application built with modern web technologies.

## How to run this project locally

1. **Clone the repository**
   ```sh
   git clone <YOUR_GIT_URL>
   ```

2. **Navigate to the project directory**
   ```sh
   cd child-safety-tracker
   ```

3. **Install dependencies**
   ```sh
   npm install
   ```

4. **Start the development server**
   ```sh
   npm run dev
   ```

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

You can deploy this project using any standard static site hosting service like Vercel, Netlify, or GitHub Pages.
